# Stok Asistanı

C# Windows Forms ve SQL Server kullanılarak geliştirilmiş
stok ve satış yönetim sistemi.

## Özellikler
- Ürün ekleme
- Ürün silme
- Güncelleme işlemleri
- Satış yönetimi
- Dashboard sistemi
- SQL veritabanı bağlantısı
- Arama filtreleme sistemi

## Kullanılan Teknolojiler
- C#
- Windows Forms
- SQL Server
- GitHub

- satislar
